Course Outline

Course Description:

Before taking this course, students should already know how to create a basic webpage using HTML5. In this course, students will learn CSS (Cascading Style Sheets) language and apply CSS to modify the appearance of the website. At the end of the course, the improved website will be deployed and published to the world.

Learning Outcomes:

Upon successful completion of this subject, the student will be able to:

  1. Understand CSS (Cascading Style Sheets) language fundamentals.
  2. Learn CSS syntax structures.
  3. Grasp the concepts of selector, property, and value.
  4. Demonstrate proficiency with styling on text, background, and other web elements.
  5. Understand and apply the concept of box model.
  6. Display page content as needed.
  7. Design and implement fundamental webpage layouts, such as one-column, two-column, and three-column layout.
  8. Design and implement simple to relatively complex navigation menus, such as one-level, multi-level parallel or vertical navigation menus.
  9. Design, develop, and deploy a website like a professional with proper styling using CSS.
