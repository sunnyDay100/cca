Welcome to Canadian Campus Awareness Inc. (CCA)

We are currently offering the following three (3) courses: Web1(HTML5), Web2(CSS), and Web3(JavaScript). Course descriptions are available below.

This site demonstrates the technologies and skills you'll learn from these courses. The purpose is to give you some idea what you can do after finishing each course.

Absolutely, you will be capable of designing, developing, and implementing more beautiful and exciting websites after finishing the courses.

Web 1: Create Your First Website with HTML5

This is an introductory web course for beginners. Students will learn Internet fundamentals, such as web terms, protocols, and client/server model. Students will learn to set up the development environment to start building their first websites using HTML5, the Hypertext Markup Language. At the end of the course, the website will be deployed and published to the world.

Web 2: Beautify Your Website with CSS

Before taking this course, students should already know how to create a basic webpage using HTML5. In this course, students will learn CSS (Cascading Style Sheets) language and apply CSS to modify the appearance of the website. At the end of the course, the improved website will be deployed and published to the world.

Web 3: Add Interactivity to Your Website with JavaScript

Before taking this course, students are expected to have knowledge of HTML5 and CSS. In this course, students will learn JavaScript as an Object-Oriented Programming (OOP) language. JavaScript basics (e.g., data types, variables, expressions, and operators) will be introduced alongside fundamental programming constructs (sequence, selection, and iteration). JavaScript functions and objects will be explored, and the DOM (Document Object Model) will be discussed as well. JavaScript will be applied to webpages to add interactivity and implement client-side validation.
