Course Outline

Course Description:

Before taking this course, students are expected to have knowledge of HTML5 and CSS. In this course, students will learn JavaScript as an Object-Oriented Programming (OOP) language. JavaScript basics (e.g., data types, variables, expressions, and operators) will be introduced alongside fundamental programming constructs (sequence, selection, and iteration). JavaScript functions and objects will be explored, and the DOM (Document Object Model) will be discussed as well. JavaScript will be applied to webpages to add interactivity and implement client-side validation.

Learning Outcomes:

Upon successful completion of this subject, the student will be able to:

  1. Understand JavaScript programming language fundamentals.
  2. Demonstrate proficiency of JavaScript basics.
  3. Grasp fundamental programming constructs, such as sequence, selection, and iteration.
  4. Understand and build JavaScript functions.
  5. Grasp the concept and create JavaScript objects.
  6. Understand and apply DOM (Document Object Model) concept to add web interactivity.
  7. Design algorithms to solve basic programming problems.
  8. Create solutions that combine web programming technologies including HTML5, CSS, and JavaScript.
